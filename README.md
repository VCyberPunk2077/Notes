# Notes

**Notes** — Android-приложение для создания, хранения и организации личных заметок. Пользователь может добавлять текст и изображения, редактировать существующие записи, искать заметки и закреплять важные из них.

---

## О проекте

Приложение реализовано на **Kotlin** с использованием **Jetpack Compose** и архитектурного подхода **Clean Architecture + MVVM**.

Данные заметок сохраняются локально через Room. Изображения, добавленные в заметки, копируются во внутреннее хранилище приложения, поэтому остаются доступными после перезапуска приложения.

---

## Основные возможности

* 📝 **Создание заметок** с заголовком и текстовым содержимым.
* ✏️ **Редактирование заметок**: изменение заголовка, текста и списка вложений.
* 🖼️ **Добавление изображений** из галереи устройства.
* 🗑️ **Удаление заметок** вместе со связанными локальными изображениями.
* 🔍 **Поиск по заметкам**: поиск выполняется по заголовку и текстовому содержимому.
* 📌 **Закрепление важных заметок** в списке.
* 📅 **Отображение даты последнего изменения** заметки.
* 💾 **Локальное хранение данных** без необходимости подключения к сети.

## Архитектура

Проект разделён на логические слои согласно принципам **Clean Architecture**:

```text
presentation → domain ← data
```

* **presentation** — UI-слой: Compose-экраны, ViewModel, навигация, состояние экранов и тема приложения.
* **domain** — бизнес-логика: сущности, интерфейсы репозиториев и use cases.
* **data** — работа с источниками данных: Room, DAO, мапперы, файловое хранилище изображений и реализация репозитория.
* **di** — Hilt-модули для предоставления зависимостей.

ViewModel отвечает за обработку пользовательских событий и подготовку состояния интерфейса. UI подписывается на `StateFlow` с использованием lifecycle-aware API.

## Установка и запуск

### Требования

Для запуска проекта потребуется:

* Android Studio;
* Android SDK;
* JDK, совместимая с используемой версией Android Gradle Plugin;
* устройство или эмулятор с Android **5.0 Lollipop (API 21)** или выше.

### Клонирование репозитория

```bash
git clone https://github.com/<your-username>/Notes.git
```

Перейдите в директорию проекта:

```bash
cd Notes
```

### Запуск через Android Studio

1. Откройте Android Studio.
2. Выберите **Open**.
3. Укажите папку с клонированным проектом.
4. Дождитесь завершения Gradle Sync.
5. Выберите устройство или эмулятор.
6. Нажмите **Run** или используйте сочетание клавиш `Shift + F10`.

### Сборка из командной строки

Для сборки debug-версии приложения выполните:

```bash
./gradlew assembleDebug
```

APK будет доступен по пути:

```text
app/build/outputs/apk/debug/app-debug.apk
```
